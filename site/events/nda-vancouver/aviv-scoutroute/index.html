AVIV ScoutRoute Guide

AVIV ScoutRoute Guide

This page contains information about installing and using the AVIV ScoutRoute mobile application at the Clark County Walk/Roll Event.

Installation

This section covers the installation of the AVIV ScoutRoute mobile app for both Android and iOS users.

Step 1: Register

  1. First, scan the QR code below or open the following link:

    https://portal.tdei.us/register?code=NDAV26HIJ

    Note: If you already have a TDEI Portal account, skip the next step. Instead, scroll to the bottom of the page and select "Sign in"

  2. Next, register for a TDEI Portal account by entering your information into the signup form

    You must enter a First Name, Email Id, and Password, then agree to the Terms and Conditions. All other fields are optional.

    Password requirements: Minimum of 8 characters in length; requires at least one lower case letter, one upper case letter, one special character, and a number.

    After entering your information, select "Create Account"

  3. After successfully creating your account, you will advance to a page that says "Registration Complete. Click Continue to proceed." Select the "Continue" button.

Step 2: Install

Installation instructions are different for Android and iOS users. If you use an Apple device, skip ahead to the iOS section.

Android

  1. On the next screen, titled "Continue in our App", select "Install Android App"

  2. This will open a new page: "Firebase App Distribution". Enter your Email and then select the "Sign up" button.

  3. After entering your Email, the page should update to say "You're in!"

  4. Within the next few minutes, you should receive an email titled "Confirm your email to test AVIV ScoutRoute for Android".

Accept the invitation

  1. Open the email on your mobile device.

  2. Select "Get started"

  3. Sign in with your Google account.

  4. Check the checkbox then select "Accept invitation"

Download and install

  1. Select "Download" (not "Download App Tester"!)

  2. A popup will appear, saying "Download file? (136 MB) app.apk". Select "Download"

  3. Once downloaded, a popup will appear, saying "Install this app? AVIV ScoutRoute". Select "Install"

  4. Once installed, a popup will appear, saying "App installed: AVIV ScoutRoute". Select "Open"

  5. Once opened, a popup will appear, saying "Allow AVIV ScoutRoute to access this device's location?" Select "Precise" and "While using the app"

  6. Once allowed, a final popup will appear, saying "Allow AVIV ScoutRoute to send you notifications?" Select "Allow"

You are now ready to start using AVIV ScoutRoute! Skip ahead to the Usage section.

iOS (Apple)

  1. On the next screen, titled "Continue in our App", select "Install iOS App"

  2. An "Open in "App Store"?" popup will appear. Select "Open"

  3. The Apple App Store app will open to the AVIV ScoutRoute page.

  4. Select the "Download" button.

  5. Once the download and installation is complete, select the "Open" button.

  6. When prompted to "Allow "AVIV ScoutRoute" to use your location?", ensure "Precise: On" is selected and then select the "Allow While Using App" button.

You are now ready to start using AVIV ScoutRoute!

Usage

This section covers the use of the AVIV ScoutRoute mobile application at the Clark County Walk/Roll Event.

Select the Workspace

  1. Open the AVIV ScoutRoute app on your mobile device

  2. Log in using the email and password you registered with in Step 1.

  3. On the home screen, you will see a list of workspaces. Select "NDA Vancouver"

    Using VoiceOver, the screen will say "AVIV ScoutRoute: Pick the workspace you want to contribute to" with a list of workspaces.

    Note: Because you registered using the event referral code, you have been automatically added to the correct project group (SCLIO Vancouver) and the correct workspace (NDA Vancouver) should be visible.

  4. The app will now display available quests in the Vancouver, WA area!

Understanding Quests

Quests in AVIV ScoutRoute are small pieces of information to gather about the built environment. Each quest type is represented by a circular icon on the map:

Icon Quest Type Description
Bus Bus Stops Quests about a bus stop location
Pedestrian inside pathway Sidewalks Covers a stretch of the pedestrian sidepath along the road
Pedestrian on marked crossing Crossings Pertains to a crossing that intersects with vehicle traffic
Traffic light Intersections Questions about a given intersection as a whole
Car Driveway Points Location of a sidewalk-driveway intersection

Tip: Quest icons stack up to keep the map easy to see. Make sure you zoom in to see every available quest!

Finding Quests On Location

When you are physically present at the event location:

  1. Ensure location services are enabled on your device

  2. The app will automatically show quests near your current location

  3. Travel along the route and tap on quest icons as you encounter them

  4. Answer the questions based on your direct observations of the environment

Downloading Data for Remote Access

Note: When opening the NDA Vancouver workspace from outside of Vancouver, WA, a message may pop up saying "Download error. Send an error report to the developer?" This is expected, as AVIV ScoutRoute is intended for use during on-the-ground, at-location surveys. Click "Cancel" and move the map to the Vancouver, WA area before continuing.

If you want to preview quests remotely (not on-location) or preload data (for example, on WiFi before beginning the walkabout, instead of using mobile data):

  1. Open the app, log in, and select the "NDA Vancouver" workspace

  2. Navigate the map to the Vancouver, WA area (from Clark College to Washington Street)

  3. Select the Menu in the top bar

  4. Select "Download data here"

  5. The app will download all quest data for the visible area on your screen

Completing a Quest

  1. Tap on a quest icon to open the quest details

  2. Read the question and observe the feature being asked about

  3. Provide your answer using one of the following input types:

    • Single Select: Choose one answer that best represents your observation
    • Numeric Entry: Enter a number using the on-screen keypad (e.g., sidewalk width in inches)
    • Multi-Select: Select one or more applicable options
    • Free Form Text: Enter a brief text response

  4. Review your answer to ensure accuracy

  5. Tap "Submit" to send the information

Once submitted, the quest will disappear from the map.

Hiding and Unhiding Elements

If you want to temporarily skip a quest:

  1. Tap on the quest icon to open it

  2. Select "Hide this" to remove it from the map temporarily

To restore hidden quests:

  1. Open Settings > Manage Quests

  2. View the Hidden Elements section

  3. Tap "Unhide All" or select individual elements to restore

Undoing Changes

If you made a mistake:

  1. Tap the Undo button

  2. Review the list of your recent edits

  3. Select an edit and tap Revert to undo it
