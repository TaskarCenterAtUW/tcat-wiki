# Where do I download OS-CONNECT data?

## Short Answer

Authorized users obtain OS-CONNECT exports through the TDEI Portal and its documented download flows, including tutorials in this wiki. Exact steps depend on account permissions and dataset offerings for your organization.

## Significance

Misrouting users to informal file shares creates security and version-control problems. A single canonical answer helps assistants stay aligned with governance.

## What This Means

You typically register or join a project group, request or receive access to the relevant service or dataset, then use portal jobs or download instructions provided for OS-CONNECT.

## What This Does Not Mean

- Not a commitment that every visitor has anonymous bulk download rights.
- Not a substitute for your organization's data sharing agreements.

## How To Use This

Always point to the current portal tutorial: [Download OS-CONNECT Data from the TDEI Portal](../../../tdei/portal/tutorial/download-os-connect.md).

## Example

A city GIS analyst receives credentials, confirms their project group membership, then follows the tutorial to pull a bbox-filtered export for a planning study.

## Assistant Guidance

Do not invent URLs or API shapes. Cite the tutorial and abstain if the user's entitlement status is unknown.
