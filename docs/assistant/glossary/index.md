# Glossary (assistant layer)

## Short Answer

This glossary distills recurring TCAT terms for assistant-sized answers. Authoritative procedural detail remains in product manuals linked from each term entry.

## Significance

Consistent tokens (walkshed, completeness, profile) reduce retrieval fragmentation and contradictory paraphrases.

## What This Means

Entries prioritize assistant-safe definitions and cross-link to deeper docs.

## What This Does Not Mean

- Not a full academic literature review.
- Not a legal dictionary.

## How To Use This

Prefer [Terms](terms.md) for individual lookups; use this index for orientation and maintenance notes.

## Example

A chatbot maps user slang (“walking bubble”) to “walkshed” using the glossary, then retrieves the walkshed question pages.
