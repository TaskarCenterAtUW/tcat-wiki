# Glossary terms (A–Z selection)

## Short Answer

**Completeness** — Modeled pedestrian network coverage for an area relative to QA/QC expectations; not ADA compliance.  
**Connected pedestrian graph** — The routable walk network used by tools, under specific rules and attributes.  
**Mobility profile** — Bundled routing assumptions in AccessMap; not a medical classification.  
**OS-CONNECT** — Washington State connected pedestrian network dataset family served via TDEI-aligned workflows.  
**TDEI** — Transportation Data Equity Initiative infrastructure for hosting, validating, and sharing datasets.  
**Walkshed** — Reachable area from an origin under a pedestrian cost budget and scenario settings.
