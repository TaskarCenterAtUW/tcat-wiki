# Use AccessMap for public engagement

## Short Answer

Use AccessMap live or via screenshots to show how pedestrian network attributes change suggested routes for different mobility profiles. Ground demos in "what the data models today" language.

## Significance

Visual routing cuts through jargon when facilitators set expectations about data limits.

## What This Means

1. Pick example trips meaningful to the community.
2. Show the same trip under two profiles or preference sets.
3. Log feedback through official channels for follow-up mapping.
4. Document questions you cannot answer on the spot for staff retrieval.

## What This Does Not Mean

- Not a referendum outcome predictor.
- Not proof that a proposed project will feel safe to every participant.

## How To Use This

Pair demos with printed links to wiki manuals and feedback pages.

## Example

At a light rail station workshop, residents compare a ramp-friendly route to a shorter stair-heavy suggestion, prompting a prioritized curb ramp inventory discussion.

## Assistant Guidance

Suggest facilitators pre-test demo addresses to avoid dead air from data gaps.
