## How To: Download OS-CONNECT Data from the TDEI Portal

This how-to walks you through downloading OpenSidewalks-formatted OS-CONNECT datasets from the TDEI Portal.

_For a list of all guides on the TCAT Wiki, refer to the [Guides List](../../../guides-list/index.md)._

---

### Prerequisites

Before you begin, you will need a TDEI Portal account. If you don't have one yet, follow the [TDEI Portal Account Registration Guide](../user-manual/account-registration.md) to create one.

---

### Step 1: Log In

1. Visit [https://portal.tdei.us/](https://portal.tdei.us/)

2. Enter your **Email Id** and **Password**

3. Select **Sign In**

![TDEI Portal login page](../../../resources/images/tdei-portal/how-to/download-os-connect/login-light.png#only-light)
![TDEI Portal login page](../../../resources/images/tdei-portal/how-to/download-os-connect/login-dark.png#only-dark)

---

### Step 2: Navigate to Datasets

After logging in, you will land on the **Dashboard**. The Dashboard displays your Project Group, Role, and Project Id, along with your API Key.

![TDEI Portal Dashboard](../../../resources/images/tdei-portal/how-to/download-os-connect/dashboard-light.png#only-light)
![TDEI Portal Dashboard](../../../resources/images/tdei-portal/how-to/download-os-connect/dashboard-dark.png#only-dark)

1. Select **Datasets** from the left sidebar

2. Select the **All Released Datasets** tab to browse all publicly available datasets

3. Use the **Type** dropdown to filter by dataset type (e.g., **OSW**)

---

### Step 3: Filter Datasets

Select the **Filter** button to narrow down the dataset list. You can filter by:

| Filter            | Description                                        |
|:------------------|:---------------------------------------------------|
| **Dataset**       | Search by dataset name                             |
| **Project Group** | **Filter by `TCAT_WSP_PG`**                        |
| **Service**       | Filter by service name                             |
| **Dataset ID**    | Search by a specific dataset ID                    |
| **Valid From**    | Filter by the start of the dataset's validity date |
| **Valid To**      | Filter by the end of the dataset's validity date   |

![Datasets filter panel](../../../resources/images/tdei-portal/how-to/download-os-connect/filter-light.png#only-light)
![Datasets filter panel](../../../resources/images/tdei-portal/how-to/download-os-connect/filter-dark.png#only-dark)

!!! info "Filter by "**Project Group: TCAT_WSP_PG**" to search for OS-CONNECT datasets."

---

### Step 4: Download a Dataset

1. Locate the dataset you want to download in the list

2. Select the **action menu** ( â‹® ) on the right side of the dataset row

3. Select **Download** from the menu

![Dataset action menu with Download option](../../../resources/images/tdei-portal/how-to/download-os-connect/download-light.png#only-light)
![Dataset action menu with Download option](../../../resources/images/tdei-portal/how-to/download-os-connect/download-dark.png#only-dark)

---

### Step 5: Choose Format and Download

1. In the **OSW Dataset Download** dialog, use the **Select Format** dropdown to choose your preferred file format (defaults to OSW)

2. Select **Download** to save the dataset to your computer

![OSW Dataset Download dialog](../../../resources/images/tdei-portal/how-to/download-os-connect/format-light.png#only-light)
![OSW Dataset Download dialog](../../../resources/images/tdei-portal/how-to/download-os-connect/format-dark.png#only-dark)

Your dataset will begin downloading. The file will be saved to your browser's default download location.
