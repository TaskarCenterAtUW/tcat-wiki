## Feedback

This section explains how to view and manage feedback submitted for datasets in the TDEI Portal.

_For a list of all guides on the TCAT Wiki, refer to the [Guides List](../../../guides-list/index.md)._

---

### Overview

The **Feedback** page lets Points of Contact view and manage feedback that has been submitted against datasets in their project group.

![Feedback page](../../../resources/images/tdei-portal/user-manual/feedback/feedback-light.png#only-light)
![Feedback page](../../../resources/images/tdei-portal/user-manual/feedback/feedback-dark.png#only-dark)

The page displays four summary cards at the top:

| Card                 | Description                                                         |
|:---------------------|:--------------------------------------------------------------------|
| **Total Feedback**   | The total number of feedback submissions received                   |
| **Turn Around Time** | The average time to resolve feedback items                          |
| **Overdue**          | The number of feedback items that have exceeded their response time |
| **Open Issues**      | The number of feedback items that remain unresolved                 |

---

### Filtering Feedback

Use the following controls to search and filter the feedback list:

| Control                        | Description                                                |
|:-------------------------------|:-----------------------------------------------------------|
| **Search Feedback By Dataset** | Search for feedback by dataset name                        |
| **Status**                     | Filter by feedback status (All Status, or specific states) |
| **Submitted After**            | Show only feedback submitted after a given date            |
| **Submitted Before**           | Show only feedback submitted before a given date           |
