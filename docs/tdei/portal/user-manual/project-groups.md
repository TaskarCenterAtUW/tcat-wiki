## Project Groups

This section explains how to view and navigate your project groups in the TDEI Portal.

_For a list of all guides on the TCAT Wiki, refer to the [Guides List](../../../guides-list/index.md)._

---

### Overview

The **Project Groups** page lists all project groups you belong to and shows which one is currently active. Use the **Search Project Group** field to filter the list by name.

![Project Groups page](../../../resources/images/tdei-portal/user-manual/project-groups/project-groups-light.png#only-light)
![Project Groups page](../../../resources/images/tdei-portal/user-manual/project-groups/project-groups-dark.png#only-dark)

Each entry displays the project group's name and its unique **Id**. Select the copy icon to copy it to your clipboard.

---
