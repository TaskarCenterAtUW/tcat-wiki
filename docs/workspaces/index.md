## Workspaces

[Workspaces](https://workspaces.sidewalks.washington.edu/) (officially, "TDEI Workspaces") is a web platform for editing TDEI datasets.

### Editors

There are multiple ways of submitting edits to data in an open workspace.

#### [AVIV ScoutRoute](../aviv-scoutroute/index.md)

AVIV ScoutRoute (formerly called [GoInfoGame](https://tcat.cs.washington.edu/goinfogame-app/)) is a volunteerism app based on [StreetComplete](https://github.com/streetcomplete/StreetComplete) that leverages the power of crowdsourcing to gather critical pedestrian and accessibility information.

#### [Rapid](../rapid/index.md)

[Rapid](https://rapideditor.org/) is an extended version of the OpenStreetMap iD editor. A modified version of Rapid is embedded in Workspaces and can be used to edit data in open workspaces.

#### [JOSM](../josm/index.md)

The [Java OpenStreetMap Editor](https://josm.openstreetmap.de/) (JOSM) is a powerful, feature-rich desktop OpenStreetMap editor.
