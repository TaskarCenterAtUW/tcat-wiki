# Content Golden Samples

These files are the canonical structural references for TCAT Wiki content pages. Use them when creating new pages, reviewing an edited page, or building an agent or prompt that needs a stable comparison target.

Use the sample that most closely matches the page you are working on:

- [topic-index.md](topic-index.md) for topic landing pages such as `docs/accessmap/index.md` or `docs/workspaces/index.md`
- [tutorial.md](tutorial.md) for single-page tutorials focused on one task or workflow
- [user-manual-index.md](user-manual-index.md) for user manual landing pages with a table of contents
- [subpage.md](subpage.md) for tutorial and user manual subpages that begin with `This section...`

For a multi-page tutorial index, use the opening pattern from [tutorial.md](tutorial.md) together with the table-of-contents pattern from [user-manual-index.md](user-manual-index.md).

These samples are intentionally generic. Reuse their structure, pacing, and formatting conventions, but do not copy the sample text into published docs without replacing placeholders and verifying the underlying content.

When comparing a page against a golden sample, check these points first:

- frontmatter and tag conventions
- page heading and required opening sentence
- Guides List reference placement
- heading hierarchy and section naming
- step formatting, bold UI labels, and monospaced literal values
- placement of screenshots, examples, and admonitions
